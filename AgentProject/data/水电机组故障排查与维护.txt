# 水电机组故障排查与运行维护技术手册

## 一、水电机组概述

水电机组是将水能转换为电能的核心设备，由水轮机、发电机、调速系统、励磁系统、冷却系统、轴承系统、辅助设备等组成。大型水电机组单机容量可达700MW至1200MW。我国主要水电机组类型包括混流式、轴流式、贯流式、冲击式和抽水蓄能式。

典型大型混流式水电机组参数：
- 额定水头：80m至200m
- 额定转速：75r/min至150r/min
- 转轮直径：5m至10m
- 额定效率：不小于95%（水轮机），不小于98.5%（发电机）
- 飞逸转速：不大于额定转速的1.65倍

## 二、运行状态监测参数

### 2.1 振动监测（关键指标）
- 上导摆度：正常运行值不大于0.20mm，报警值0.30mm，停机值0.40mm
- 下导摆度：正常运行值不大于0.20mm，报警值0.30mm，停机值0.40mm
- 水导摆度：正常运行值不大于0.25mm，报警值0.35mm，停机值0.45mm
- 上机架振动：正常运行值不大于0.06mm，报警值0.10mm
- 下机架振动：正常运行值不大于0.06mm，报警值0.10mm
- 顶盖振动：正常运行值不大于0.05mm，报警值0.08mm
- 振动频率特征：转频振动分量不超过通频值的70%

### 2.2 温度监测
- 推力轴瓦温度：正常运行值55℃至70℃，报警值75℃，停机值80℃
- 上导轴瓦温度：正常运行值50℃至65℃，报警值70℃，停机值75℃
- 下导轴瓦温度：正常运行值50℃至65℃，报警值70℃，停机值75℃
- 水导轴瓦温度：正常运行值45℃至60℃，报警值65℃，停机值70℃
- 定子铁芯温度：不超过120℃（B级绝缘），报警值110℃
- 定子绕组温度：不超过130℃（F级绝缘），报警值120℃
- 冷却水温升：冷却器进出口温差不超过15K

### 2.3 压力与流量监测
- 蜗壳进口压力：正常运行MPa值在设计值的±5%范围内
- 尾水管出口压力：不小于0.02MPa（防止汽蚀）
- 主轴密封水压力：0.15MPa至0.25MPa
- 技术供水总管压力：0.3MPa至0.6MPa
- 导叶漏水量：关闭状态下不超过额定流量的0.2%

### 2.4 电气参数监测
- 发电机出口电压：额定电压18kV至20kV，允许偏差±5%
- 定子电流三相不平衡度：不大于8%
- 励磁电压：正常范围100V至400V
- 轴电流：不大于0.5A（正常），报警值1A
- 局部放电量：不超过5000pC（在线监测参考值）

### 2.5 气蚀与磨损监测
- 转轮气蚀面积：不超过叶片总面积的5%
- 气蚀深度：最大坑深不超过3mm
- 导叶间隙：上下端面间隙0.15mm至0.25mm，立面间隙不大于0.10mm
- 止漏环间隙：单边间隙增大不超过设计值的150%

## 三、常见故障与排查方法

### 3.1 机组振动过大
故障现象：上导/下导/水导摆度超过报警值，机组运行有明显异常噪音。
原因排查：
1. 质量不平衡：转子或转轮质量不平衡，需进行动平衡校正（平衡精度G2.5级）
2. 轴线不正：盘车检查轴线摆度，折线值不超过0.02mm/m
3. 电磁不平衡：定子三相电流不平衡、转子匝间短路
4. 水力不平衡：导叶开度不一致、转轮叶片型线偏差、尾水管涡带
5. 轴承间隙过大：导轴瓦间隙超过设计值0.05mm以上
6. 基础松动：基础螺栓预紧力不足，共振频率与转频接近

处置方案：
1. 频谱分析确定振动主频，区分机械、电磁、水力原因
2. 变转速试验（30%至100%额定转速）确定共振区
3. 变励磁试验确定电磁振动分量
4. 变负荷试验确定水力振动分量
5. 根据分析结果制定针对性检修方案

### 3.2 轴瓦温度异常升高
故障现象：某导轴瓦温度持续升高超过报警值。
原因排查：
1. 润滑油劣化：油质化验黏度下降超过10%，酸值超过0.2mgKOH/g
2. 冷却系统故障：冷却水量不足，冷却器传热管结垢
3. 轴瓦间隙不当：间隙过小导致油膜无法正常建立
4. 轴电流：绝缘垫损坏导致轴电流通过轴瓦
5. 瓦面损伤：瓦面巴氏合金局部磨损或脱胎

处置方案：
1. 油质化验分析，酸值>0.3mgKOH/g时更换新油（L-TSA46透平油）
2. 检查冷却水流量和水质，清洗冷却器传热管
3. 轴瓦间隙测量与调整，导轴瓦单边间隙一般0.15mm至0.25mm
4. 轴电流监测，检查绝缘垫电阻（不低于1MΩ）
5. 瓦面研刮修复，接触面积不小于75%

### 3.3 水轮机出力不足
故障现象：在相同水头和导叶开度下，机组出力低于预期值。
原因排查：
1. 转轮气蚀或磨损导致水力效率下降
2. 导叶间隙过大导致漏水量增加
3. 尾水管涡带导致水力损失增加
4. 拦污栅堵塞导致水头损失增加（拦污栅前后水位差超过2m）
5. 引水隧洞衬砌破损导致水头损失

处置方案：
1. 转轮效率试验（热力学法或流速仪法），效率下降超过2%考虑检修
2. 导叶间隙测量与调整
3. 尾水管补气装置检查，补气量调节
4. 拦污栅清污机运行频率调整
5. 引水隧洞放空检查

### 3.4 发电机定子绝缘劣化
故障现象：绝缘电阻下降，局部放电量增大。
原因排查：
1. 定子绕组端部电晕放电
2. 定子铁芯松动导致槽楔脱落
3. 绕组受潮或油污污染
4. 热老化导致绝缘材料性能下降

处置方案：
1. 定子绝缘电阻测试：极化指数不小于2.0（R10min/R1min）
2. 介质损耗角测试：tanδ增量（Δtanδ）不大于0.5%（0.2Un至0.6Un）
3. 局部放电在线监测系统安装与校准
4. 绕组端部防晕处理：涂覆低阻防晕漆（面电阻率5kΩ至20kΩ）
5. 槽楔紧度检查：松动的槽楔重新打紧或更换
6. 定子绕组清洗干燥（真空干燥温度85℃±5℃，时间不小于48小时）

### 3.5 调速系统故障
故障现象：机组转速或负荷调节不稳定，响应时间超标。
原因排查：
1. 电液伺服阀卡涩或零漂
2. 主配压阀位移传感器故障
3. 油质劣化导致控制油路堵塞
4. PID参数设置不当

处置方案：
1. 调速器静特性试验：转速死区不大于0.02%，非线性度不大于1%
2. 甩负荷试验：用25%、50%、75%、100%额定负荷进行，转速上升不超过调节保证值
3. 油过滤或更换：油液清洁度不低于NAS 7级
4. 电液伺服阀清洗或更换，零位调整
5. PID参数在线优化整定

## 四、计划性检修周期

### 4.1 A级检修（大修，周期8至10年）
- 发电机转子抽出，定子全面检查
- 水轮机转轮拆出检修，气蚀补焊修复
- 导水机构全部解体检查
- 轴承全部翻瓦检查
- 调速系统全面校验
- 励磁系统全面检修

### 4.2 B级检修（中修，周期4至5年）
- 轴承检查及轴瓦研刮
- 主轴密封更换
- 导水机构部分解体
- 冷却器清洗及水压试验
- 电气预防性试验

### 4.3 C级检修（小修，每年1次）
- 轴承油质化验及过滤
- 导叶间隙测量
- 振动摆度全面测量分析
- 自动化元件校验
- 滤水器清洗

### 4.4 D级检修（临检，根据运行状态确定）
- 消除运行中发现的缺陷
- 更换易损件（碳刷、滤芯、密封圈）
- 一般性清扫检查
